# Student Performance Analysis and Prediction

This project predicts student performance using Machine Learning based on features like study hours, attendance, and previous scores. It helps identify students at risk and supports data-driven decision making.

## Features
- Data analysis using Python
- Machine Learning model for prediction
- Flask web application
- Real-time prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python app.py
